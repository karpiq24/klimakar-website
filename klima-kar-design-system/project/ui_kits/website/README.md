# Klima-Kar Website UI Kit

## Overview
Hi-fidelity recreation of the Klima-Kar informational website.
Single-page scroll with: top bar → nav → hero → services grid → authorised partners → contact → footer.

## Design Width
1280px desktop, responsive down to 375px mobile.

## Files
- `index.html` — Interactive prototype (loads all JSX components)
- `Header.jsx` — Top bar + navigation
- `Hero.jsx` — Hero section with CTA
- `ServicesGrid.jsx` — Service cards grid
- `Partners.jsx` — Authorised partner logos/section
- `ContactSection.jsx` — Contact details + booking form
- `Footer.jsx` — Site footer

## Notes
- Barlow / Barlow Condensed loaded from Google Fonts
- Lucide icons via CDN
- No real map; placeholder used in ContactSection
- Logo files loaded from ../../assets/
